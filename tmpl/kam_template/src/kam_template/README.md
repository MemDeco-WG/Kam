# Template Module

This is a Kam template module.

## Variables

- `{{name}}`: Module name
- `{{version}}`: Module version
- `{{author}}`: Module author
- `{{description}}`: Module description
- `{{id}}`: Module ID

## Usage

Use this template to create new Kam modules by implementing it with `kam init --impl <path_to_this_dir>`.

## Recursive Beauty

This template can be used to create other templates, allowing for meta-template creation.
